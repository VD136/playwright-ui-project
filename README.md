# Playwright UI Automation Project

## Description
End-to-end **web UI automation** using **Playwright**.  
Automates functional flows, form validations, and cross-browser interactions.  
Tests are integrated with **CI/CD** to run automatically on every push or pull request.

---

## Features
- Cross-browser testing: **Chromium, Firefox, WebKit**  
- Page Object Model (POM) for reusable components  
- Dynamic data handling for forms and UI interactions  
- Optional field and conditional validation  
- **HTML and Allure reports** for detailed test insights  
- GitHub Actions CI/CD workflow for **real-time feedback**

---

## Tech Stack
- **Test Framework:** Playwright Test  
- **Languages:** JavaScript / TypeScript  
- **Reporting:** HTML + Allure  
- **CI/CD:** GitHub Actions  
- **Browsers:** Chromium, Firefox, WebKit
